Turples
A Tuple in TypeScript is a special type of array that has a fixed number of elements, where each element has a specific type. 
It allows you to define an ordered list with known types for each position.

Enums
An Enum (short for "enumeration") is a way to assign names to a set of numeric or string values.
It makes code more readable.



This is useful when we want to group different types if value together. It works for a fixed array
